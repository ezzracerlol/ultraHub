# ultraHub
Hi, this is a special hub where you can find apps for tracking and customizing the performance of fpv drones. PLEASE DONT COPY MY PROJECT. email: ezzracerlol@gmail.com
For first use: open terminal and paste this command py -m pip install PyQt6 PyQt6-WebEngine pyserial opencv-python pyqtgraph numpy
Check the releases tab and download the desired application in zip format, unpack it, and run the exe file
